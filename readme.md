# **API REST utilizando Hibernate, Wildfly e PostgreSQL**

## **PASSOS DE INSTALAÇÃO**

1. Clone o repositório utilizando seu cliente git favorito ou extraia o arquivo .zip.

2. No administrador do postgresql, cria o banco de dados de nome "produtos".

3. Configure na aplicação as credenciais para acessar o banco dados local. Usuário padrão para esse projeto: postgres; Senha padrão: 123456.

4. Abra o diretório do projeto através do menu Abrir Projeto do NetBeans.

5. Adicione o servidor wildfly no menu de serviços do NetBeans;

6. Nas propriedades do projeto, em Run, defina o servidor WildFly adicionado no passo 5, 

7. Clique em compilar e executar ou use o atalho do teclado.
